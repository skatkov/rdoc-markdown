# Class Bird
The base class for all birds.

### Public Instance Methods

<a id="method-i-fly"></a>
#### `fly(direction: string, velocity: number) -> bool`
Fly somewhere.

Flying is the most critical feature of birds.

##### Example

```
fly(:south, 70)
```

<a id="method-i-speak"></a>
#### `speak() { |text| ... }`
Produce some noise.
